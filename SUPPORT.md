# Support Learn JavaScript

If this repository helps you learn JavaScript, there are several simple ways to support it.

## Star the Repository

Starring the repository helps more learners discover it through GitHub search, recommendations, and topic pages.

## Share the Project

You can help by sharing the repository with:

- JavaScript learners
- Students and self-taught developers
- Beginner programming communities
- Web development groups
- Friends looking for JavaScript notes or a beginner-friendly JavaScript course

## Open Useful Issues

Helpful issues include:

- Confusing explanations
- Incorrect examples
- Broken links
- Missing JavaScript topics
- Typos or formatting issues
- Ideas that make the learning path clearer

## Contribute Improvements

Contributions are welcome.

Please remember:

- Open pull requests against `develop`
- Do not open pull requests directly against `main`

For contribution guidelines, see:

- [CONTRIBUTING.md](./CONTRIBUTING.md)

## Sponsor the Work

If you want to support the project financially, you can sponsor the work through GitHub Sponsors:

- https://github.com/sponsors/jaganganesh

Sponsorship helps fund ongoing chapter writing, documentation improvements, and the future expansion of the repository.

## Other Helpful Support

You can also support the project by:

- Watching the repository for updates
- Forking the project
- Referencing it in articles, videos, newsletters, or learning lists
- Sharing it in study groups, bootcamps, or developer communities
- Suggesting improvements that make the repository easier to discover and use

Every star, share, issue, contribution, and sponsorship helps the project grow.
